FocusFlow 🌿
A Modern Pomodoro Timer & Productivity Growth App
FocusFlow is a minimal, student-friendly productivity tool built to help you stay focused, track your progress, and grow your "Focus Plant" through consistent study sessions. It combines the classic Pomodoro technique with modern gamification and ambient soundscapes.
✨ Key Features
🎯 Smart Pomodoro Timer: Fully customizable study and break durations with automatic session transitions.
🌱 Growth System: Earn "Focus Points" as you complete sessions and watch your digital plant grow from a seedling to a blooming flower.
🚫 Focus Lock (Distraction Blocker): A simulated app-blocking overlay that triggers during study sessions to keep you in the zone.
📊 Deep Analytics: Weekly activity charts, daily study time tracking, and a streak counter to keep you motivated.
🎵 Ambient Soundscapes: Built-in focus sounds (Soft Rain, Ocean Waves, Morning Forest) to drown out distractions.
💬 Bilingual Motivation: Random motivational quotes in both English and Bangla after every session.
🌙 Modern Design: A high-performance dark-themed UI built with Glassmorphism, smooth Framer Motion animations, and responsive layouts.
🛠️ Tech Stack
Framework: React 19 + TypeScript
Styling: Tailwind CSS 4
Animations: Motion (formerly Framer Motion)
Charts: Recharts
Icons: Lucide React
Development: Vite

Notifications: FocusFlow requests standard Browser Notifications to alert you when a session ends.
